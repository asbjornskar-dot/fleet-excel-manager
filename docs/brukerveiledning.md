# Fleet Excel Manager – Brukerveiledning

## 1. Formål
Fleet Excel Manager er eit nettlesarbasert verktøy for å forbetre informasjonsflyt
mellom interne Excel-filer og eksterne samarbeidspartnarar.

Verktøyet erstattar ikkje interne IT-system og krev ingen integrasjonar eller tilgangar.

---

## 2. Kva verktøyet gjer
- Leser Excel-filer eksportert frå interne system
- Tolkar og standardiserer informasjon
- Samlar kommentarar og status på same bil
- Genererer nye Excel-filer til ulike formål

---

## 3. Kva verktøyet IKKJE gjer
- Skriver ikkje til interne system
- Har ikkje tilgang til databasar eller API
- Lagrar ikkje data permanent
- Endrar ikkje originalfiler

All bruk skjer manuelt via nettlesar.

---

## 4. Slik brukar du verktøyet

### Steg 1 – Eksporter Excel
Eksporter Excel-fil frå internt system som vanleg.

### Steg 2 – Last opp fil
Opne Fleet Excel Manager i nettlesar og last opp Excel-fila.

### Steg 3 – Kontroller data
Kontroller tabellen som blir vist:
- Manglande reservasjonsnummer
- Manglande VIN
- Kommentarar

### Steg 4 – Last ned filer
Last ned genererte filer:
- Bilpleiehallen.xlsx – ekstern bruk
- Full_oversikt.xlsx – intern oversikt
- Admin.xlsx – kontroll og korrigering

---

## 5. Kvar skal filene brukast

| Fil | Bruk |
|----|----|
Bilpleiehallen.xlsx | Sendast til ekstern leverandør |
Full_oversikt.xlsx | Intern koordinering |
Admin.xlsx | Korrigering og kvalitetssikring |

---

## 6. Vanlege spørsmål

**Må eg installere noko?**  
Nei. Alt skjer i nettlesar.

**Krev dette IT-godkjenning?**  
Nei. Verktøyet brukar berre filer brukaren lastar opp sjølv.

**Blir data lagra?**  
Nei. Data eksisterer berre under aktiv bruk.

---

## 7. Viktig prinsipp
Fleet Excel Manager er eit støtteverktøy for betre informasjonsflyt,
ikkje eit nytt IT-system.
